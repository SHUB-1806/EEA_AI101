# AI-101 Winter Project

### Electronics Engineering Association (EEA), IIT Kanpur

**Duration:** December 2024 – February 2025

## Overview

This repository contains the material studied during the AI-101 Winter Project organized by the Electronics Engineering Association (EEA), IIT Kanpur.

The project focused on building a foundational understanding of modern Artificial Intelligence and Deep Learning techniques used in Computer Vision and Natural Language Processing. Alongside introductory model implementations, significant emphasis was placed on reading, analyzing, and presenting contemporary AI research papers.

This repository specifically documents the literature review component of the project and contains the papers, notes, and presentation material related to the research topics explored.

---

## Project Objectives

* Understand the fundamentals of Deep Learning and Neural Networks.
* Explore image classification using Convolutional Neural Networks (CNNs).
* Study text classification using Transformer-based architectures.
* Learn the principles behind tokenization, attention mechanisms, and language models.
* Develop the ability to read and critically analyze modern AI research papers.
* Improve technical communication through research presentations and discussions.

## Research Papers Studied

### 1. DeepFace: Closing the Gap to Human-Level Performance in Face Verification

This paper introduced one of the earliest large-scale deep learning systems capable of achieving near human-level accuracy in face verification tasks.

Key concepts explored:

* Face alignment using 3D models
* Deep neural representations for facial recognition
* Large-scale supervised learning
* Feature extraction from facial images
* Face verification pipelines
* Benchmark evaluation methodologies

### 2. CacheBlend: Fast Large Language Model Serving for RAG with Cached Knowledge Fusion

This paper focuses on improving the efficiency of Retrieval-Augmented Generation (RAG) systems by introducing mechanisms that reuse previously retrieved knowledge during inference.

Key concepts explored:

* Retrieval-Augmented Generation (RAG)
* Knowledge retrieval systems
* Cached knowledge reuse
* Efficient Large Language Model serving
* Latency reduction techniques
* Knowledge fusion strategies
* Scalability challenges in modern LLM deployments
